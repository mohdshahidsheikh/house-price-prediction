# House Price Prediction — King County, USA

Exploratory data analysis and regression modelling on a 21,613-record housing dataset.

# Tools
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook

# Results
- Best model: Polynomial Ridge Regression Pipeline
- R-squared: 0.75
- Models tested: Simple Linear, Multiple Linear, Ridge, Polynomial Regression
